## Summary

First stable release of Revia — a mobile-first spaced repetition app for personal learning. Users can sign in, import or create decks, study lessons, complete daily reviews, and track progress on the dashboard.

---

## Added

- **Auth** — Supabase email/password sign-up, sign-in, sign-out
- **Dashboard** — Due today, reviewed today, streak, deck/card counts, recent decks
- **Decks** — List, create, delete, detail view
- **Lessons** — Create, delete, tap-to-study with swipe card viewer (normal + reverse mode)
- **Review** — Daily due queue, reveal + rate 1–5, spaced repetition scheduler (`simple-v1`)
- **Search** — Find decks, lessons, and cards
- **Settings** — Theme (light/dark), JSON import, create deck, account
- **Import** — JSON deck import (file upload or paste)
- **Mobile shell** — Floating nav, full-screen study/review, desktop mobile-first gate

## Infrastructure

- Vercel production deploy (`revia` root directory)
- Supabase Postgres + Auth (`ap-south-1`)
- Vercel functions region `bom1` (Mumbai)
- Prisma with connection pooling

## Changed

- Performance: optimistic review UI, query prefetching, optimized dashboard/deck queries
- API auth: session cache, no per-request user DB sync on hot path

## Known limitations (not blockers)

- Deck/lesson edit UI not exposed (API exists)
- Card management UI not on deck page (API exists)
- No export, statistics page, or tags UI
- No image/audio upload on cards

## Migration notes

None — first public release. New users sign up via Supabase; data stored in Supabase Postgres.
