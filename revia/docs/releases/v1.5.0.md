## Summary

Revia now supports **renaming decks and lessons** in the UI, **safer JSON import** (confirm before import, reliable large decks), and **faster startup**. Sample **Telugu** and **Kannada** deck JSON files are included in the repo for import testing.

---

## Added

- **Deck title edit** — Pencil icon on deck detail page (`InlineTitleEditor`)
- **Lesson title edit** — Pencil icon next to each lesson title on deck detail
- **Sample language decks** — `content/decks/kannada-basics.json` (301 cards) and `content/decks/telugu-basics.json` (302 cards) with Telugu/Kannada on front, English on back, romanization in parentheses
- **Import generators** — `generate-kannada-basics.py` and `generate-telugu-basics.py` for regenerating sample decks

## Changed

- **Import flow** — Uploading a file loads JSON into the text area; import runs only when you click **Import JSON**
- **Large imports** — Import service batches one transaction per lesson (avoids timeouts on 300+ card decks)
- **Import cache** — Invalidates deck detail and lesson queries after import
- **About page copy** — Describes Practice + Daily Review (replaces old Review-first wording)

## Fixed

- **Slow app load** — Removed global practice cards prefetch on every page load
- **Explore spinner** — Fixed infinite loading when background refetch runs
- **Post-import errors** — Large deck imports no longer fail mid-transaction as often; destination dropdown respected before import

---

## Import format reminder

- **Front:** Language script + `(romanization)` — e.g. `నమస్కారం (namaskaram)`
- **Back:** English
- No hard cap on card count; very large single imports may hit platform limits — split into multiple decks if needed
