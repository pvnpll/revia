## Summary

Public decks can be **added to your library** with full study and review progress. Original authors stay credited on imported copies. Settings adds **Feedback** for suggestions and bug reports, and **Username** moves under **Account**.

---

## Added

- **Import public decks** — `Add to library` on Explore decks; copies lessons and cards into your private library with fresh scheduling
- **Author attribution** — Imported decks show `Originally by @username` on deck detail and in your deck list
- **Import API** — `POST /api/decks/:deckId/import`
- **Feedback** — Settings → Feedback; submit suggestions or bug reports (`POST /api/feedback`)
- **Feedback storage** — `feedback` table in Postgres

## Changed

- **Account settings** — Username editor moved inside Account (removed separate Username hub item)
- **Public deck detail** — Import card replaces read-only-only messaging; preview still available before import

## Database

- `decks.source_deck_id`, `decks.source_author_username` — tracks imported copies
- `feedback` table — stores user feedback
